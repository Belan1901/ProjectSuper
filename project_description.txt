Задача: автоматически определить кол-во ядер на каждом изображении, применив для этого открытые python библиотеки для работы с изображениями (например, torch или OpenCV).
Полученный результат представим в ноутбуке в виде таблицы Pandas, где каждой строке соответствует одна картинка, а колонке - какой-то параметр (к примеру, кол-во ядер на слайде). Помимо колонки с кол-вом объектов добавим колонки со средним размером ядра на слайде, медианным размером ядра, средним радиусом и средней интенсивностью свечения ядра (интенсивности соответствует значение пикселя - чем больше оно, тем выше яркость).
На основе этой таблицы построим графики в том же ноутбуке при помощи библиотеки matplotlib (или другой, при желании). На графиках отразим разницу между картинками по кол-ву объектов, размерам и интенсивности свечения объеков.
